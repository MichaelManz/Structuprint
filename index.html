NAME

Structuprint

DESCRIPTION

A utility for analysis of MD simulations (and single PDB structures as well), offering a novel 2D visualisation technique. Its output may be a single frame or a full-blown animation, depending on input.

DEPENDENCIES

-the Perl interpreter, >= 5.10

-Astro::MapProjection, >= 0.01 (Perl module)

-Bio::PDB::Structure, >= 0.01 (Perl module)

-DBD::SQLite, >= 1.37 (Perl module)

-DBI, >= 1.622 (Perl module)

-File::Spec, >= 3.33 (Perl module)

-File::Copy, >= 2.18 (Perl module)

-Gtk2, >= 1.247 (Perl module) [for GNU/Linux only]

-List::Util, >= 1.25 (Perl module)

-Math::Round, >= 0.06 (Perl module)

-Statistics::R, >= 0.30 (Perl module)

-the R interpreter, >= 2.15.1

-ggplot2, >= 0.9.3 (R package)

-ImageMagick

-Gifsicle

-xterm [for GNU/Linux only]
